# Mentor Demo Script

1. Open the overview page and introduce Data Agent Sandbox as a portfolio-safe public demo.
2. Explain the product lifecycle preview: Question, Topic, Metric, SQL Plan, Validation, Trace, Evaluation, and Report.
3. Open Retail Growth Demo and show the topic header, information card, source overview, glossary, and topic health metadata.
4. Click "What was total revenue last week?" and run it from the composer.
5. Show the final answer, generated SQL, validation checks, result rows, KPI chart, trace timeline, guardrail decision, and suggested follow-ups.
6. Run "Why did revenue drop last week?" and point out the diagnostic caveat: the answer is directional and not a causal conclusion.
7. Open Experiment Metrics Demo and run "Did the latest week have complete data?" to show warnings and a needs-review guardrail decision.
8. Enter "Export all customer emails and rank risky users." to show that sensitive user-level export is blocked before SQL generation.
9. Explain the confidentiality boundary: synthetic data only, no internal data, code, prompts, schemas, screenshots, metrics, roadmap details, or proprietary workflows.
10. Explain the next stage: evaluation dashboard, bad-case review, richer editable reports, and optional LLM integration only behind explicit API-key configuration.
