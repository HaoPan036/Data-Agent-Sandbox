# 导师演示脚本

1. 打开 overview 页面，介绍 Data Agent Sandbox 是一个适合作品集展示的公开 demo。
2. 说明产品生命周期预览：Question、Topic、Metric、SQL Plan、Validation、Trace、Evaluation 和 Report。
3. 打开 Retail Growth Demo，展示 Topic header、信息卡、数据源概览、术语表和 topic health 元数据。
4. 点击一个示例问题，展示它会填入底部输入框。
5. 说明本阶段刻意不执行查询：下一阶段会接入确定性 intent routing、SQL generation、validation、本地 execution、trace logging 和 grounded answers。
6. 说明保密边界：只使用合成数据，不包含内部数据、代码、提示词、schema、截图、指标、路线图细节或专有工作流。
7. 说明为什么下一阶段回答 "Does it work?"：它会把示例问题连接到经过校验的 SQL、执行结果、trace、evaluation checks 和报告草稿。
