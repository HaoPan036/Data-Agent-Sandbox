# Confidentiality Boundary

**This project uses synthetic or public data only. It does not contain internal company data, code, prompts, schemas, screenshots, business metrics, roadmap details, or proprietary workflows.**

The frontend, Node serverless API, deterministic agent contracts, UI layout, topics, questions, schemas, metrics, and reports in this repository were created as public demo material. The visual structure uses common analytics-product conventions and does not reproduce a private product or workflow.

## Included

- Deterministically generated ecommerce, traffic, campaign, product, masked-customer, refund, and experiment-event data.
- Public frontend and serverless backend source code created for this repository.
- Generic topic metadata, semantic schemas, metric definitions, knowledge entries, sample questions, guardrails, and evaluation cases.
- Public documentation, tests, CI configuration, and deployment configuration for the sandbox.

## Excluded

- Internal or confidential company datasets and business metrics.
- Internal code, private prompts, private schemas, private screenshots, dashboards, feature names, or roadmap details.
- Credentials, customer systems, private warehouses, or workflows that require access to them.
- Claims that the demo represents a specific company's production architecture or proprietary process.

## Future Data Connections

Any future connector in this sandbox may access public datasets only. Connector support will not broaden the repository's synthetic-or-public-data-only boundary.
