# Confidentiality Boundary

This project uses synthetic or public data only. It does not contain internal company data, code, prompts, schemas, screenshots, business metrics, roadmap details, or proprietary workflows.

This UI layout is a public original layout inspired by common analytics platform patterns. It does not copy internal branding, internal data, internal topic names, internal screenshots, internal feature names, internal schemas, or proprietary workflows.

## Included

- Synthetic ecommerce, traffic, campaign, product, masked customer, refund, and experiment event data.
- Public open-source frontend code created for this repository.
- Generic semantic metadata, metric definitions, knowledge entries, topics, sessions, and sample questions.
- Public documentation about the sandbox architecture and confidentiality boundary.

## Excluded

- Internal company datasets.
- Internal code or private system prompts.
- Internal schemas, screenshots, dashboards, metrics, roadmap details, feature names, or proprietary workflows.
- Any workflow that requires access to private systems.
