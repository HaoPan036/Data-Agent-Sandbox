# 保密边界

**本项目只使用合成数据或公开数据。项目不包含任何公司内部数据、代码、提示词、Schema、截图、业务指标、路线图细节或专有工作流。**

English boundary statement: **This project uses synthetic or public data only. It does not contain internal company data, code, prompts, schemas, screenshots, business metrics, roadmap details, or proprietary workflows.**

本仓库中的前端、Node Serverless API、确定性 Agent 契约、UI 布局、Topics、问题、Schemas、指标和报告，都是专门创建的公开演示材料。视觉结构只采用常见分析产品惯例，不复刻任何私有产品或工作流。

## 包含内容

- 以确定性方式生成的电商、流量、活动、商品、脱敏客户、退款和实验事件数据。
- 为本仓库创建的公开前端和 Serverless 后端源代码。
- 通用 Topic 元数据、语义 Schema、指标定义、知识条目、示例问题、Guardrails 和评估 Cases。
- Sandbox 的公开文档、测试、CI 配置和部署配置。

## 不包含内容

- 任何公司内部或保密数据集和业务指标。
- 内部代码、私有提示词、私有 Schema、私有截图、仪表盘、功能名称或路线图细节。
- 凭据、客户系统、私有数仓，或任何需要访问这些资源的工作流。
- 声称本 Demo 代表某家公司的生产架构或专有流程。

## 未来数据连接

本 Sandbox 未来增加的 Connector 只能访问公开数据集。Connector 不会扩大本仓库“只使用合成数据或公开数据”的边界。
