# Project Plan

## Version 0.1

- Create a Vite React TypeScript app.
- Add synthetic ecommerce data.
- Implement deterministic intent routing.
- Generate validated read-only SQL.
- Execute SQL in the browser.
- Render result tables and charts.
- Record trace events.
- Run a deterministic evaluation suite.
- Create editable HTML report drafts.

## Later Versions

- Add more public or synthetic datasets.
- Add more business-question templates.
- Strengthen SQL parsing and validation.
- Export reports.
- Add optional LLM support behind explicit user configuration.
